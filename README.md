# SocialCat

SocialCat は X（x.com / twitter.com）を使いやすくする Chrome 拡張です。

## 主な機能
- 返信は `Enter` で送信（デフォルト ON）
- 通常投稿は設定で `Enter` 送信を有効化可能（デフォルト OFF）
- `Shift + Enter` は改行
- いいね後に自動でリプ欄を開く（デフォルト OFF）
- リポスト投稿を折りたたむ（デフォルト ON）
- リプ送信時に自動でいいね（デフォルト OFF）
- 指定時間帯に挨拶を自動挿入（複数時間帯、ランダム対応）

## 設定画面
- ポップアップ: 機能ごとの ON/OFF を切り替え
- オプションページ: 時間帯・挨拶メッセージの詳細設定

## インストール（開発者モード）
1. Chrome で `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」を ON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダ（`manifest.json` がある場所）を選択
